From Zero to Live

The story of setting up a full GitHub workflow — entirely from the terminal. No shortcuts, one command at a time.

01 Today's Mission

Go from a blank machine to a live, shareable webpage — using nothing but the command line.

✔ Installed Git

Got version control running on Windows via winget, then verified it worked.

$ git --version
git version 2.55.0.windows.3

✔ Created an SSH Key

Generated an ed25519 key pair and added the public key to GitHub for secure, password-free access.

$ ssh-keygen -t ed25519 -C "your@email.com"
The key fingerprint is: SHA256:h7GP...77sE

✔ Connected to GitHub

Tested the SSH handshake and got the green light.

$ ssh -T git@github.com
Hi joyeeta-max! You've successfully authenticated.

✔ Set Up the GitHub CLI

Installed gh and logged in, so repos can be managed without leaving the terminal.

$ gh auth login
✓ Logged in as joyeeta-max

✔ Cloned & Rebuilt This Page

Cloned this repository as a collaborator, redesigned it, and pushed it back — where you're reading it now.

$ gh repo clone anishsaha-cell/first-page
Receiving objects: 100% ... done.

02 The Takeaway

Every step above happened in a terminal — the same tools professional developers use every day. What started as a blank screen is now a live website anyone in the world can open.
